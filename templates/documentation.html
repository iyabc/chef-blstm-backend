POST /api/prediction

get_prediction()

To get a prediction, make a POST request to the following endpoint: /api/prediction

Request Body: 
    {
        "ingredients": [List of ingredients separated by commas (e.g. selectedNERs)]
    }
Response: 
    {
        "prediction": {
          title: string (e.g. Chicken Adobo);
          ingredients: string[] (e.g. ['5 pcs chicken breasts', '1 cup soy sauce','1/2 cup vinegar', etc.]); 
          instructions: string[] (e.g. ['chop garlic and onion', 'add salt and pepper to chicken','sear the chicken in a hot oil', etc.]);
          ner_tags: string[] (e.g. ['chicken breasts', 'soy sauce','vinegar', etc.]);
        }
    }
